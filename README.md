# Easy C++ Games

A collection of simple console-based games written in C++.  
This repository is made for beginners who want to practice programming concepts through small and easy projects.

## Games Included

- Number Guessing Game
- Rock Paper Scissors
- Tic Tac Toe

---

## About

These games are written using basic C++ concepts such as:

- Loops
- Functions
- Conditional statements
- Arrays
- Random number generation

The code is kept simple and readable so that beginners can understand and modify it easily.

---


## Games

### Number Guessing Game

A simple game where the player guesses a random number between 1 and 10.

### Rock Paper Scissors

Play Rock, Paper, Scissors against the computer.

### Tic Tac Toe

A basic two-player Tic Tac Toe game played in the terminal.
